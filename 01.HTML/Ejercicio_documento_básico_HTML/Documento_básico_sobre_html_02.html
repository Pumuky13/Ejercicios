Etiquetas HTML


El lenguaje HTML está conformado por un sistema de etiquetas en serie o tags, que incluyen instrucciones que los navegadores traducen como:

  • Imágenes
  • Texto
  • Hipervínculos
  • Listas de palabras
  • Tablas, etc...

Así, para que un elemento de este tipo pueda ser leído, la etiqueta debe tener dos partes: una etiqueta de inicio <...> y una de cierre </...> entre guiones < >. Por ejemplo, si en nuestro documento HTML escribimos una frase con las etiquetas <strong> y </strong> definen un texto en negrita.
